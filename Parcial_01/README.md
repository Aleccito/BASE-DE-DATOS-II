## Descripción del Proyecto

El presente proyecto corresponde al Parcial #1 de la asignatura Base de Datos II. Su propósito es el diseño, implementación y documentación de una base de datos de carácter empresarial, aplicando buenas prácticas de ingeniería de software y administración de bases de datos (DBA).

---

## Objetivos

1. Diseñar modelos entidad–relación (DER) con cardinalidades correctas.
2. Aplicar normalización hasta la Tercera Forma Normal (3FN).
3. Implementar la base de datos en un contenedor Docker.
4. Utilizar sentencias DDL, DML y operaciones CRUD.
5. Crear vistas SQL orientadas a negocio, seguridad y auditoría.
6. Documentar las decisiones técnicas adoptadas en el rol de DBA.

---

## Implementación Técnica

La base de datos se despliega mediante un contenedor Docker utilizando el motor relacional MariaDB/MySQL, garantizando portabilidad y reproducibilidad del entorno de desarrollo.
