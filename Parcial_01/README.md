## 📌 Descripción del Proyecto

Este proyecto corresponde al **Parcial #1 de Base de Datos II**, donde se desarrolla el diseño, implementación y documentación de una base de datos empresarial utilizando buenas prácticas de ingeniería de software y administración de bases de datos (DBA).


---

## 🎯 Objetivos del Proyecto

- Diseñar modelos entidad–relación (DER) complejos con cardinalidades correctas.
- Aplicar normalización hasta Tercera Forma Normal (3FN).
- Implementar bases de datos en contenedores Docker.
- Utilizar DDL, DML y operaciones CRUD.
- Crear vistas SQL para negocio, seguridad y auditoría.
- Documentar decisiones técnicas como DBA.

---

## 🐳 Implementación con Docker

Se utiliza un contenedor Docker con motor de base de datos relacional (MariaDB/MySQL).
